### Reporting Issues

If you find incorrect information:

1. **Check if the issue already exists** in the [Issues](https://github.com/talhaXdev/awesome-b2b-lead-generation/issues) section
2. **Create a new issue** with:
   - Clear title describing the problem
   - URL of the affected entry
   - Description of what's incorrect
   - Suggested correction (if known)

### Suggesting New Entries

We welcome suggestions for new agencies and tools! To suggest an addition:

1. **Open an issue** with the label `new-entry`
2. **Provide the following information**:
   - Company/Tool name
   - Website URL
   - Category (e.g., Email Outreach, LinkedIn Automation)
   - Brief description
   - Pricing range (if available)
   - Why it should be included

### Improving Descriptions

Help us improve existing entries:

- Fix typos or grammatical errors
- Update outdated information
- Add more detailed descriptions
- Include additional resources or documentation
